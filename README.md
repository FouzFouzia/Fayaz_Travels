Fayaz Travels is a Core Java console-based application developed using Object-Oriented Programming (OOP) principles.
The project manages driver information such as category, driver ID, driver name, and total distance travelled using the List interface and ArrayList implementation.

This project demonstrates clean architecture, modular coding, and practical usage of Java Collections.
Project Structure

📂 Driver.java
→ Model class containing driver attributes and getter/setter methods

📂 Travel.java
→ Contains business logic methods for data processing

📂 Test.java
→ Main class to execute and test the application

List vs ArrayList 

List is an interface in Java.

ArrayList is a class that implements the List interface.

Using List reference (List<Driver>) improves flexibility.

ArrayList provides dynamic resizing and built-in methods like add(), remove(), size().

This follows the principle: "Program to an interface, not an implementation."
